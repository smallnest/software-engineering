# Design: Snake Game (贪吃蛇)

## Problem Statement
单文件贪吃蛇网页游戏，纯前端，零依赖。ch04 版本的迭代升级：增加移动端触控、视觉效果、最高分记录。

## What Makes This Cool
- 移动端触控（滑动控制方向，点击开始）
- 吃食物时粒子爆发效果
- 蛇身运动残影拖尾
- 食物光晕、蛇身渐变色
- 蛇头有眼睛，跟随移动方向
- 速度随得分递增（130ms → 55ms）
- 最高分 localStorage 持久化
- 中文 UI 提示
- 单文件，零依赖，打开即玩

## Technical Specification

### 画布
- 网格：20×20，单元格 28px
- 画布：560×560px

### 游戏状态
- 蛇初始长度：3 节，位置 [4,10], [5,10], [6,10]
- 初始方向：RIGHT
- 初始速度：130ms/tick
- 加速步进：每吃一个食物 -3ms
- 最低速度：55ms/tick
- 初始状态：暂停，显示"贪吃蛇 — 按空格或点击开始"

### 食物
- 随机生成在空白格子，均匀分布
- 渲染：红色圆点 r=10.6px，带径向渐变光晕

### 碰撞检测
- 撞墙：头坐标超出 [0, COLS) × [0, ROWS)
- 撞自身：头与蛇身重叠（吃食物时排除尾部即将移出的格子）

### 蛇渲染
- 每节：圆角矩形 (roundRect, r=7px)，带径向渐变体光
- 颜色：HSB 色相从 140→170 渐变（头绿尾偏青）
- 头部有白色眼睛 + 黑色瞳孔，瞳孔偏移跟随移动方向

### 粒子效果
- 吃食物时生成 8 个粒子
- 径向分布，速度 2.5px/frame
- 色相随机 30-60（黄绿区间），饱和度 90%，亮度 55%
- 生命周期 1.0，每帧衰减 0.03

### 运动残影
- 每步在蛇头位置生成一个残影点
- 生命周期 1.0，每帧衰减 0.04
- 渲染：rgba(46,204,113, alpha) 圆点 r≈9.8px

### 输入
- 键盘：Arrow keys 控制方向，Space 开始/重开
- 触控：滑动方向判定（最小滑动距离 20px，主轴优先）
- 点击画布：开始/重开

### 最高分
- Key: `snake_high_v2`，localStorage 存取
- private browsing 下 localStorage 不可用时静默降级

### 性能
- requestAnimationFrame 主循环
- 60fps 目标，粒子/残影数组每帧清理过期元素
- 移动端 touch-action: none 防止页面滚动

## Constraints
- 单 HTML 文件（内联 CSS/JS）
- Vanilla JS，无框架无构建
- Canvas 2D 渲染
- 键盘 + 触控双输入

## Premises
1. 单文件，零依赖 — AGREE
2. Canvas 2D 渲染 — AGREE
3. 键盘 + 触控双输入 — AGREE
4. Vanilla JS — AGREE
5. 比 ch04 版本增强 — AGREE

## Approaches Considered

### Approach A: ch04 clone + touch
ch04 结构加触控，最小改动。
Effort: S. Risk: Low.

### Approach B: Full rewrite with visual effects (CHOSEN)
全新代码，粒子、残影、渐变、眼睛、速度递增、最高分。
Effort: M. Risk: Low.

### Approach C: Game framework (Phaser/PixiJS)
引入第三方库。
Effort: M. Risk: Med. 违反零依赖约束。

## Recommended Approach
**Approach B** — 全重写。视觉 delight 是 builder mode 的核心目标。

## Edge Cases
- localStorage 不可用（隐私模式）：静默降级，最高分功能跳过
- 极小视口（<320px）：画布固定 560px，需滚动
- 食物占满全屏无空位：spawnFood 返回 [0,0] 防崩溃
- 快速连续滑动：每次 touchend 取最终 delta，无队列
- Tab 切换：requestAnimationFrame 自动暂停，无需处理
- 同时触控+键盘：两者独立，无冲突
